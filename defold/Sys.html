staticload (filename:String):AnyTable

Load a lua table from a file on disk.

If the file exists, it must have been created by Sys.save to be loaded.

staticload_resource (filename:String):String

Loads a custom resource from game data.

Specify the full filename of the resource that you want to load. In order for the engine to include custom resources in the build process, you need to specify them in the "game.project" settings file.

staticopen_url (url:String):Bool

Open url in default application (typically a browser).

Returns a boolean indicating if the url could be opened or not.

staticsave (filename:String, table:AnyTable):Bool

Saves a lua table to a file stored on disk.

The table can later be loaded by Sys.load. Use Sys.get_save_file to obtain a valid location for the file.

Returns a boolean indicating if the table could be saved or not.

staticset_connectivity_host (host:String):Void

Set host to check for network connectivity against.

staticset_error_handler (error_handler:String ‑> String ‑> String ‑> Void):Void

Set the error handler (a function which is called whenever a lua runtime error occurs). Handler arguments: source, message, traceback.
