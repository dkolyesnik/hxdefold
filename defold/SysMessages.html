class SysMessages

package defold

import defold.Sys

Messages handled by the system (send to the @system: socket).

Static variables

staticread onlyExit:Message<{code:Int}> = new Message<{ var code : Int}>("exit")

Exits application.

Terminates the game application and reports the specified code to the OS.

staticread onlyReboot:Message<SysMessageReboot> = new Message<SysMessageReboot>("reboot")

Reboots engine with arguments.

Arguments will be translated into command line arguments. Sending the reboot command is equivalent to starting the engine with the same arguments.

staticread onlySetUpdateFrequency:Message<{frequency:Int}> = new Message<{ var frequency : Int}>("set_update_frequency")

Sets update frequency.

This option is equivalent to display.update_frequency but set in run-time.

staticread onlyStartRecord:Message<SysMessageStartRecord> = new Message<SysMessageStartRecord>("start_record")

Starts video recording of the game frame-buffer to file.

staticread onlyStopRecord:Message<Void> = new Message<Void>("stop_record")

Stop current video recording.

staticread onlyToggleProfile:Message<Void> = new Message<Void>("toggle_profile")

Shows/hides the on-screen profiler.
