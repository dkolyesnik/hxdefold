class Render

package defold

Rendering functionality to be used from render scripts.

See RenderMessages for messages related to render scripts.

Static methods

staticclear (buffers:Table<RenderBufferType, EitherType<Vector4, Float>>):Void

Clears the active render target.

Clear buffers in the currently enabled render target with specified value.

staticconstant_buffer ():RenderConstantBuffer

Create a new constant buffer.

Constant buffers are used to set shader program variables and are optionally passed to the Render.draw function.

staticdelete_render_target (render_target:RenderTarget):Void

Deletes a render target.

Disables the currently enabled material.

staticdisable_render_target (render_target:RenderTarget):Void

Disables a render target.

staticdisable_state (state:RenderState):Void

Disables a render state.

staticdisable_texture (unit:Int, render_target:RenderTarget):Void

Disables a texture for a render target.

staticdraw (predicate:RenderPredicate, ?constants:RenderConstantBuffer):Void

Draws all objects matching a predicate.

An optional constants buffer can be provided to override the default constants. If no constants buffer is provided, a default system constants buffer is used containing constants as defined in materials and set through *.set_constant() and *.reset_constant() on visual components.
