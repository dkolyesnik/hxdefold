class Go

package defold

Manipulation of game objects and core hooks for Lua script logic.

See GoMessages for standard game object messages.

Static methods

staticanimate (url:HashOrStringOrUrl, property:HashOrString, playback:GoPlayback, to:EitherType<Vector3, EitherType<Quaternion, Float>>, easing:EitherType<GoEasing, Vector3>, duration:Float, ?delay:Float, ?complete_function:Void ‑> Void):Void

Animates a named property of the specified game object or component.

This is only supported for numerical properties. If the node property is already being animated, that animation will be canceled and replaced by the new one.

staticcancel_animations (url:HashOrStringOrUrl, property:HashOrString):Void

Cancels all animations of the named property of the specified game object or component.

By calling this function, all stored animations of the given property will be canceled.

staticdelete (?id:HashOrStringOrUrl):Void

Deletes a game object instance.

If the id of the instance to delete is not specified, the instance of the calling script is deleted.

staticdelete_all (?ids:Table<Int, Hash>):Void

Deletes a set of game object instance.

Delete all game objects simultaneously as listed in table. The table values (not keys) should be game object ids (hashes).

staticget (url:HashOrStringOrUrl, id:HashOrString):GoProperty

Gets a named property of the specified game object or component.

staticget_id (?path:String):Hash

Gets the id of an instance.

The instance id is a hash of the absolute path. If path is specified, it can either be absolute or relative to the instance of the calling script. If path is not specified, the id of the instance of the calling script will be returned.

staticget_position (?id:HashOrStringOrUrl):Vector3

Gets the position of the instance.

The position is relative the parent (if any). Use Go.get_world_position to retrieve the global world position.

staticget_rotation (?id:HashOrStringOrUrl):Quaternion

Gets the rotation of the instance.

The rotation is relative to the parent (if any). Use Go.get_world_rotation to retrieve the global world position.

staticget_scale (?id:HashOrStringOrUrl):Float

Gets the uniform scale factor of the instance.

The uniform scale is relative the parent (if any). Use Go.get_world_scale to retrieve the global world scale factor.

staticget_scale_vector (?id:HashOrStringOrUrl):Vector3

Gets the 3D scale factor of the instance.

The scale is relative the parent (if any). Use Go.get_world_scale to retrieve the global world scale factor.

staticget_world_position (?id:HashOrStringOrUrl):Vector3

Gets the instance world position.

Use Go.get_position to retrieve the position relative to the parent.

staticget_world_rotation (?id:HashOrStringOrUrl):Quaternion

Gets the instance world rotation.

Use Go.get_rotation to retrieve the rotation relative to the parent.

staticget_world_scale (?id:HashOrStringOrUrl):Float

Gets the instance world scale factor.

Use Go.get_scale to retrieve the scale factor relative to the parent.

staticset (url:HashOrStringOrUrl, id:HashOrString, value:GoProperty):Void

Define a property to be used throughout the script.

This function defines a property which can then be used in the script through the self-reference. The properties defined this way are automatically exposed in the editor in game objects and collections which use the script. Note that you can only use this function outside any callback-functions like init and update.

staticset_position (position:Vector3, ?id:HashOrStringOrUrl):Void

Sets the position of the instance.

The position is relative to the parent (if any). The global world position cannot be manually set.

staticset_rotation (rotation:Quaternion, ?id:HashOrStringOrUrl):Void

Sets the rotation of the instance.

The rotation is relative to the parent (if any). The global world rotation cannot be manually set.

staticset_scale (rotation:EitherType<Float, Vector3>, ?id:HashOrStringOrUrl):Void

Sets the scale factor of the instance.

The scale factor is relative to the parent (if any). The global world scale factor cannot be manually set. NOTE! Physics are currently not affected when setting scale from this function.
