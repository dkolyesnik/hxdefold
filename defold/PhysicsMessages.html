class PhysicsMessages

package defold

import defold.Physics

Messages related to collision object component.

Static variables

staticread onlyApplyForce:Message<{position:Vector3, force:Vector3}> = new Message<{ var force : Vector3; var position : Vector3}>("apply_force")

Applies a force on a collision object.

Post this message to a collision-object-component to apply the specified force on the collision object. The collision object must be dynamic.

staticread onlyCollisionResponse:Message<{other_position:Vector3, other_id:Hash, group:Hash}> = new Message<{ var other_id : Hash; var other_position : Vector3; var group : Hash}>("collision_response")

Reports a collision between two collision objects.

This message is broadcasted to every component of an instance that has a collision object, when the collision object collides with another collision object. For a script to take action when such a collision happens, it should check for this message in its on_message callback function.

This message only reports that a collision actually happened and will only be sent once per colliding pair and frame. To retrieve more detailed information, check for the ContactPointResponse instead.

staticread onlyContactPointResponse:Message<PhysicsMessageContactPointResponse> = new Message<PhysicsMessageContactPointResponse>("contact_point_response")

Reports a contact point between two collision objects.

This message is broadcasted to every component of an instance that has a collision object, when the collision object has contact points with respect to another collision object. For a script to take action when such contact points occur, it should check for this message in its on_message callback function.

Since multiple contact points can occur for two colliding objects, this message can be sent multiple times in the same frame for the same two colliding objects. To only be notified once when the collision occurs, check for the CollisionResponse message instead.

staticread onlyRayCastResponse:Message<PhysicsMessageRayCastResponse> = new Message<PhysicsMessageRayCastResponse>("ray_cast_response")

Reports a ray cast hit.

This message is sent back to the sender of a Physics.ray_cast, if the ray hit a collision object.

staticread onlyRequestVelocity:Message<{linear_velocity:Vector3, angular_velocity:Vector3}> = new Message<{ var linear_velocity : Vector3; var angular_velocity : Vector3}>("request_velocity")

Requests the velocity of a collision object.

DEPRECATED! Read properties linear_velocity and angular_velocity with Go.get instead.

Post this message to a collision-object-component to retrieve its velocity.

staticread onlyTriggerResponse:Message<{other_id:Hash, group:Hash, enter:Bool}> = new Message<{ var other_id : Hash; var enter : Bool; var group : Hash}>("trigger_response")

Reports interaction (enter/exit) between a trigger collision object and another collision object.

staticread onlyVelocityResponse:Message<PhysicsMessageContactPointResponse> = new Message<PhysicsMessageContactPointResponse>("velocity_response")

Reports the velocity of a collision object.

DEPRECATED! Read properties linear_velocity and angular_velocity with Go.get instead.

This message is sent back to the sender of a RequestVelocity message.
